# ParseHubBot
---

Telegram聚合解析Bot, 支持AI总结视频和图文

支持的平台:
`抖音视频|图文`、`哔哩哔哩视频|动态`、`YouTube`、`YouTube Music`、`TikTok视频|图文`、`小红书视频|图文`、`Twitter视频|图文`、`百度贴吧图文|视频`、`Facebook视频`、`微博视频|图文`......
